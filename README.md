## PRO lab 3 (варіант 20)

Множення матриць на структурі **2D решітка** засобами **MPI**.

- **Варіант 20**: решітка **4×2**, \(n1=87\), \(n2=319\), \(n3=213\)
- Обчислюємо: \(C[n1][n3] = A[n1][n2] \times B[n2][n3]\)
- MPI запуск: **1 master (rank 0) + 8 workers (rank 1..8)** → всього **9 процесів**

### Збірка

Потрібні `mpicc`/`mpirun` (наприклад OpenMPI).

```bash
make
```

### Налаштування розмірності матриць

**Послідовна програма** `./sequential`:

| Виклик | Розміри |
|--------|--------|
| `./sequential` | 87×319×213 (варіант 20, scale=1) |
| `./sequential <scale>` | (87×scale)×(319×scale)×(213×scale) |
| `./sequential <n1> <n2> <n3>` | довільні A[n1][n2], B[n2][n3], C[n1][n3] |

**MPI програма** `./mpi_grid`:

| Виклик | Розміри |
|--------|--------|
| `./mpi_grid` | 87×319×213 |
| `./mpi_grid <scale> [verbose]` | масштаб від варіанту 20 |
| `./mpi_grid <n1> <n2> <n3> [verbose]` | довільні розміри |

`verbose`: 1 — виводити повідомлення обміну, 0 — лише час.

### Запуск

```bash
# Послідовно (за замовчуванням або scale)
./sequential
./sequential 2

# Довільні розміри
./sequential 100 200 150

# MPI (9 процесів)
mpirun -np 9 ./mpi_grid 1 1
mpirun -np 9 ./mpi_grid 174 638 426 0
```

### Вимір часу

```bash
# Масштаби 1, 2, 3, 5, 10 (варіант 20)
python3 benchmark.py

# Тільки масштаби 2, 5, 10
python3 benchmark.py 2 5 10

# Один тест з заданими розмірами
python3 benchmark.py --dims 174 638 426
```

Результати в `out/timings.csv`. Матриці в `out/*.tsv` пишуться лише для малих розмірів (базовий варіант або менші).
